## 🤖 AI in Product Management (End-to-End)

| Stage | Goal | How to Use AI | Tools | Advanced Layer | Open Source / Access |
|------|------|--------------|------|----------------|----------------------|
| 🔍 Discovery | Understand problems | Summarize interviews, detect patterns, generate hypotheses | [ChatGPT](https://chat.openai.com), [Dovetail](https://dovetail.com) | Continuous feedback loops from interviews & support tickets | ChatGPT (free tier), Dovetail (paid) |
| 👤 User Research | Understand users | Cluster responses, sentiment analysis, persona creation | [Typeform](https://typeform.com), [Maze](https://maze.co) | Reduce time-to-insight (hours → minutes) | Free tier limited, mostly paid |
| 💡 Ideation | Generate solutions | Brainstorm features, benchmark, explore edge cases | [ChatGPT](https://chat.openai.com), [Miro](https://miro.com) | AI as product sparring partner | ChatGPT (free), Miro (free tier) |
| 🧠 Prioritization | Decide what to build | Simulate RICE, analyze impact, suggest trade-offs | [Airtable](https://airtable.com), [Coda](https://coda.io) | Dynamic prioritization using real-time data | Free tier limited |
| 🗺️ Strategy | Define direction | Create vision, synthesize insights, positioning | [Notion AI](https://notion.so/product/ai), [ChatGPT](https://chat.openai.com) | Connect data → strategy automatically | Notion AI (paid), ChatGPT (free) |
| 🧾 PRD / Specs | Document product | Generate PRDs, user stories, acceptance criteria | [Notion AI](https://notion.so/product/ai), [Confluence](https://atlassian.com/software/confluence) | Living documentation (auto-updated) | Mostly paid |
| 🎨 Design & UX | Create experience | Generate wireframes, UX copy, prototype ideas | [Figma](https://figma.com), [Uizard](https://uizard.io) | Predictive UX testing before build | Figma (free), Uizard (limited free) |
| 🧑‍💻 Development | Build product | Generate code, debug, build MVPs | [GitHub Copilot](https://github.com/features/copilot), [Cursor](https://cursor.sh), [Lovable](https://lovable.dev) | Faster delivery, low/no-code prototyping | Copilot (paid), Cursor (freemium), Lovable (limited free) |
| 🧪 QA / Testing | Ensure quality | Generate test cases, automate tests, detect bugs | [Playwright](https://playwright.dev), [TestRail](https://testrail.com) | Continuous QA integrated with dev | Playwright (open source), TestRail (paid) |
| 🚀 Launch | Go to market | Generate release notes, emails, campaigns | [ChatGPT](https://chat.openai.com), [HubSpot](https://hubspot.com) | Personalized messaging by segment | ChatGPT (free), HubSpot (limited free) |
| 📈 Analytics | Measure performance | Analyze funnels, detect churn, generate insights | [Mixpanel](https://mixpanel.com), [Amplitude](https://amplitude.com) | Real-time automated insights | Good free tier, advanced paid |
| 🔁 Growth & Iteration | Improve product | A/B testing ideas, optimize funnel, find opportunities | [Optimizely](https://optimizely.com), [VWO](https://vwo.com) | Continuous loop: data → insight → action | Mostly paid |
